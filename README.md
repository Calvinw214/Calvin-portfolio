# Calvin Williams III — Portfolio

Personal portfolio website for Calvin Williams III, Healthcare Data & Operations Analyst.

**Live site:** `https://<your-github-username>.github.io/` (after you publish it — see steps below)

---

## Project structure

```
portfolio/
├── index.html          # Main HTML page
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Interactivity (nav, reveal, form)
├── assets/
│   ├── calvin.jpg      # Portrait photo
│   └── logo.jpg        # Favicon / "III" logo
├── .nojekyll           # Tells GitHub Pages not to run Jekyll
└── README.md           # This file
```

## Built with

- Semantic HTML5
- Modern CSS (custom properties, grid, flexbox, clamp, backdrop-filter)
- Vanilla JavaScript (no frameworks, no build step)
- Google Fonts: Fraunces, Inter, JetBrains Mono

## Local preview

Just open `index.html` in your browser. No build step required.

Optional — run a local server for a slightly cleaner experience:
```bash
# Python 3
python -m http.server 8000
# then visit http://localhost:8000
```
